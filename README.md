# UTN-OE-Clima

Análisis de Anomalías de Temperatura Global
Trabajo Práctico — Organización Empresarial | UTN TUP 2026
Integrante
Maria Sol Savid Patoco
Escenario
Análisis de datos climáticos históricos usando el dataset GISTEMP (NASA), con registros mensuales de temperatura global desde 1850 hasta 2026.
Estructura
UTN-OE-Clima/
├── datos/          # Dataset monthly.csv
├── scripts/        # Script de análisis en Python
├── resultados/     # Gráfico e indicadores generados
├── README.md
└── .gitignore
Cómo ejecutar
python%run scripts/analisis_datos.py
El script genera automáticamente los resultados en la carpeta /resultados.
